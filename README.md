# Strix

* Clone repo or download zip.
```bash
git clone git@github.com:muszkin/strix.git
cd strix
```
* Use data.sql to create and insert sample data to db.
* Install dependencies
```bash
composer install
```
* Run tests
```bash
php bin/phpunit
```
* Run project 
```bash
php bin/console server:run
```
* Navigate to http://127.0.0.1:8000/
